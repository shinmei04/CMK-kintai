# 勤怠・請求メモ

レッスンと講師外業務を記録するためのアプリです

スマホでもPCでも使えます

記録した内容はGoogleスプレッドシートに保存されます

## このアプリでできること

- レッスンをコマ数で記録できます
- 講師外業務を出勤と退勤で記録できます
- 講師外業務の時間を自動で計算できます
- 最近の勤怠を確認できます
- 月末に集計できます
- コピー用テキストを作れます

## 使うファイル

この中で使うファイルは主に2つです

- `Code.gs`
- `Index.html`

`README.md` はこの説明書です

## 必要なもの

- Googleアカウント
- Googleスプレッドシート
- Google Chromeなどのブラウザ

難しい環境設定は不要です

## まずPCで試す

本番で使う前に、PCで画面だけ試せます

1. GitHubのページを開きます
2. 緑色の `Code` ボタンを押します
3. `Download ZIP` を押します
4. ダウンロードしたZIPファイルを開きます
5. 中にある `Index.html` をダブルクリックします
6. ブラウザで画面が開けばOKです

画面上に `PCテストモード` と出ていればテスト用です

このテストではスプレッドシートには保存されません

## 本番用の準備

ここからは実際に使うための準備です

### 1 Googleスプレッドシートを作る

1. Googleドライブを開きます
2. 左上の `新規` を押します
3. `Googleスプレッドシート` を押します
4. ファイル名を `勤怠・請求メモ` にします

### 2 Apps Scriptを開く

1. スプレッドシートの上のメニューを見ます
2. `拡張機能` を押します
3. `Apps Script` を押します

別の画面が開きます

ここにアプリの中身を入れていきます

### 3 Code.gsを貼り付ける

1. Apps Script画面の左側にある `Code.gs` を押します
2. 最初から入っている文字を全部消します
3. このGitHubにある `Code.gs` を開きます
4. 中身を全部コピーします
5. Apps Scriptの `Code.gs` に貼り付けます
6. 保存します

保存は上の保存アイコン、または `Ctrl + S` です

Macの場合は `command + S` です

### 4 Index.htmlを作る

1. Apps Script画面の左側にある `+` を押します
2. `HTML` を押します
3. 名前を `Index` にします
4. 作られた `Index.html` を開きます
5. 最初から入っている文字を全部消します
6. このGitHubにある `Index.html` を開きます
7. 中身を全部コピーします
8. Apps Scriptの `Index.html` に貼り付けます
9. 保存します

Apps Script上で名前が `Index` と表示されても問題ありません

### 5 初期設定をする

1. Apps Script画面の上にある関数選択で `setup` を選びます
2. `実行` を押します
3. Googleアカウントを選ぶ画面が出たら、自分のアカウントを選びます
4. 確認画面が出たら `詳細` を押します
5. プロジェクト名のリンクを押します
6. `許可` を押します

終わるとスプレッドシートに次のシートが自動で作られます

- `勤怠ログ`
- `月次集計`

### 6 アプリのURLを作る

1. Apps Script画面の右上にある `デプロイ` を押します
2. `新しいデプロイ` を押します
3. 種類の選択で `ウェブアプリ` を選びます
4. 説明には `初回` などと入力します
5. `次のユーザーとして実行` は `自分` を選びます
6. `アクセスできるユーザー` は自分だけで使うなら `自分のみ` を選びます
7. `デプロイ` を押します
8. 表示されたURLを開きます

このURLがアプリの入口です

スマホで使う場合は、このURLをスマホで開いてください

## 普段の使い方

### レッスンを記録する

1. `講師業務` を見ます
2. コマ数を選びます
3. 各コマの内容を選びます
4. 必要なら補足メモを書きます
5. `レッスンを保存` を押します

日付は今日の日付で保存されます

### 講師外業務を始める

1. `講師外業務` の `出勤` を押します
2. 内容を入力します
3. 分類を選びます
4. 必要ならメモを書きます
5. `出勤する` を押します

### 講師外業務を終える

1. 作業が終わったら `退勤` を押します
2. 終了時刻と時間が自動で入ります

### 間違えて出勤したとき

まだ退勤していない場合は `出勤を取り消す` を押します

退勤済みの記録を直したい場合は、スプレッドシートの `勤怠ログ` で直接直します

### 勤怠歴を見る

1. `勤怠歴` を押します
2. 最近の記録が表示されます

### 月末に集計する

1. `月次出力` を押します
2. 対象月を選びます
3. `申請用出力` を押します
4. 集計を確認します
5. `コピー用テキストを作成` を押します

コピー用テキストには講師外業務の詳細だけが入ります

## コマ内容のルール

通常コマとして数えるもの

- `CB`
- `CS`
- `MJ`
- `MS`
- `CE`
- `プラコ`
- `体験会`
- `振替`
- `補講`
- `キャンプその他`

キャンプ10時として数えるもの

- `キャンプ10時`

確認が必要なもの

- `その他・要確認`

`その他・要確認` は給与計算には入れません

## 時間の計算

講師外業務は開始時刻と終了時刻から分を計算します

時間は小数第2位まで表示します

例

- 30分は `0.50`
- 75分は `1.25`
- 10分は `0.17`

## アプリを更新するとき

GitHubの `Code.gs` や `Index.html` を新しくしただけでは、公開済みのアプリは変わりません

更新するときは次の順番で進めます

1. Apps Scriptの `Code.gs` を貼り替えます
2. Apps Scriptの `Index.html` を貼り替えます
3. 保存します
4. `デプロイ` を押します
5. `デプロイを管理` を押します
6. 鉛筆アイコンを押します
7. バージョンで `新バージョン` を選びます
8. `デプロイ` を押します

## 困ったとき

### 画面が変わらない

Apps Scriptで新しいバージョンをデプロイしてください

### 権限エラーが出る

デプロイ設定で `次のユーザーとして実行` が `自分` になっているか確認してください

### 出勤したままになった

その作業を終わらせるなら `退勤` を押します

間違えただけなら `出勤を取り消す` を押します

### 入力を間違えた

スプレッドシートの `勤怠ログ` を開き、該当する行を直接直してください

### PCテストモードの記録を消したい

一番簡単なのは別のブラウザで `Index.html` を開くことです

同じブラウザで消したい場合は、ブラウザのサイトデータを削除してください

## 最後に確認すること

本番で使う前に、次の操作を一度試してください

1. レッスンを1件保存する
2. 講師外業務で出勤する
3. 講師外業務で退勤する
4. 勤怠歴を見る
5. 月次出力をする
6. スプレッドシートの `勤怠ログ` と `月次集計` に反映されているか見る

ここまでできれば準備完了です
